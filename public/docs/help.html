Getting Started with PaleoDraw

Welcome to PaleoDraw, a specialized vector drawing application designed for creating precise paleontological illustrations and scientific diagrams.

Quick Start

  1. Create a New Project: Click File → New Project or press Ctrl+N
  2. Select a Tool: Choose from the Tools menu or use keyboard shortcuts
  3. Start Drawing: Click on the canvas to place control points and create splines
  4. Save Your Work: Press Ctrl+S to save your project

First Steps

When you first open PaleoDraw, you'll see a blank canvas with a grid. The grid helps you align your drawings precisely. You can adjust the grid size from the View menu.

💡 Tip: Start with the Curve tool (press C) to create smooth, natural-looking lines perfect for paleontological outlines.

Interface Overview

Main Components

  • Title Bar: Displays the application name and window controls
  • Menu Bar: Access to File, Edit, View, and Tools menus
  • Canvas: Main drawing area with gridlines for precision
  • Theme Toggle: Switch between light and dark modes
  • Help Button: Opens this documentation (you're here!)

Canvas Navigation

  • Pan: Hold Space and drag, or use middle mouse button
  • Zoom: Scroll wheel, or View menu options
  • Fit to Canvas: View → Fit to Canvas to reset view

Grid Settings

The grid can be customized from the View menu. Adjust grid spacing to match your needs - smaller grids for detailed work, larger grids for overview sketches.

Drawing Tools

Select Tool T

The default tool for selecting and manipulating existing objects.

  • Click on a spline to select it
  • Drag to move selected objects
  • Use resize handles to scale objects
  • Rotate using the rotation handle

Curve Tool (B-Spline) C

Creates smooth, flowing curves ideal for organic shapes.

  • Click to place control points
  • The curve automatically smooths between points
  • Press R to reverse point direction (add to beginning vs end)
  • Press Escape to finish drawing
  • Point Snapping: Hold Ctrl while dragging a point to snap it to nearby points within 15 pixels. A green indicator shows the snap target. Perfect for closing shapes and connecting splines precisely.
💡 Tip: Use fewer points for smoother curves. You can always add more points later by selecting the spline and clicking on the curve.

Line Tool L

Creates straight line segments between points.

  • Click to place points
  • Lines connect sequentially
  • Great for angular shapes and technical diagrams

Straight Tool (Nurbs) N

Creates NURBS curves for precise mathematical curves.

  • Similar to B-Spline but with different interpolation
  • Useful for technical precision

Import SVG I

Import existing SVG files into your project.

  • Select File → Import SVG or press I
  • Choose an SVG file from your computer
  • Imported objects can be moved, resized, and rotated

Delete Spline P

Quick tool for removing splines.

  • Activate the tool with P
  • Click on any spline to delete it

Project Management

Creating Projects

Start a new project from File → New Project or Ctrl+N. This creates a blank canvas ready for drawing.

Saving Projects

PaleoDraw saves projects as JSON files (.json) that contain all your splines, imported SVG objects, and canvas settings.

  • Save: Ctrl+S - Saves to current file location
  • Save As: Ctrl+Shift+S - Choose a new location/name

Opening Projects

Open existing projects in several ways:

  • File → Open Project (Ctrl+O) - Browse for a .json file
  • File → Open Recent (Ctrl+Shift+O) - Quick access to recently opened projects
  • Welcome Screen: Shows recent projects when you start the app
💡 Tip: Use the folder icon in recent projects to open the file location in your file explorer.

Exporting

Export your finished work as an SVG file (Ctrl+E) that can be opened in other vector graphics applications like Inkscape, Adobe Illustrator, or used in publications.

Importing SVG

Import existing SVG files to incorporate them into your project. Imported objects can be positioned, scaled, and rotated like any other object.

Frequently Asked Questions

How do I add more points to an existing spline?

Select the spline with the Select Tool (T), then click directly on the curve where you want to add a point. A new control point will be created at that location.

How do I delete a single point?

Select the spline, then click on the point you want to delete and press Del. You can also select multiple points by holding Ctrl and clicking each point.

Can I change the color or stroke of splines?

Currently, PaleoDraw focuses on creating clean vector paths. For styling, export your work as SVG and use a dedicated illustration program like Inkscape or Adobe Illustrator.

What's the difference between the curve tools?

  • Curve (B-Spline): Creates smooth, organic curves ideal for natural shapes
  • Line: Creates straight segments between points
  • Straight (NURBS): Creates mathematically precise curves with different interpolation

How do I reverse the direction while drawing a spline?

While actively drawing a spline, press R to toggle between adding points to the end or beginning of the spline. This is useful when you realize you need to extend the spline in the opposite direction.

My grid is too small/large. How do I change it?

Go to View menu and use the grid size input to set a custom value. Smaller values create finer grids, larger values create more spacious grids.

Can I work without the grid?

Yes! The grid is purely visual and doesn't affect your drawing. You can adjust canvas settings in the View menu.

Where are my projects saved?

Projects are saved wherever you choose when using Save As. The application also keeps track of recently opened projects for quick access.

Can I undo mistakes?

Yes! Press Ctrl+Z to undo actions, and Ctrl+Y to redo. PaleoDraw maintains a history of your recent changes.

About PaleoDraw

Version Information

Version 0.3.0

PaleoDraw is a specialized vector drawing application designed for creating precise paleontological illustrations and scientific diagrams.

Features

  • Multiple spline types (B-Spline, Line, NURBS) for different drawing needs
  • SVG import and export for integration with other tools
  • Precise control point manipulation with visual feedback
  • Point snapping (Ctrl+drag) for exact alignment
  • Grid-based alignment with customizable spacing
  • Multi-object and multi-point selection
  • Comprehensive undo/redo history (up to 50 states)
  • Dark and light themes for comfortable work
  • Project management with recent files tracking
  • Keyboard-driven workflow for efficiency
  • Drag-and-drop SVG import
  • Template library with common fossil types
  • Z-ordering controls for layering objects

Technology

Built with modern web technologies for performance and cross-platform compatibility:

  • Electron 38.4.0 - Cross-platform desktop framework
  • React 19.1.1 - Reactive user interface
  • SVG.js 3.2.5 - Vector graphics manipulation
  • Material-UI 7.3.5 - Modern interface components
  • Vite 7.1.7 - Fast build tooling

System Requirements

  • Operating System: Windows 10/11, macOS 10.15+, or Linux
  • RAM: 4 GB minimum, 8 GB recommended
  • Display: 1280x800 minimum resolution
  • Storage: 200 MB for application
